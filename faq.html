応募・面接・移住・生活に関する疑問にお答えします

応募・面接について

未経験でも応募できますか？ +
はい、未経験の方も大歓迎です。入社後の研修制度が充実しており、業務に必要なスキルは入社後に習得できます。 重要なのは、新しい環境でチャレンジする意欲と、お客様とコミュニケーションを取ることが好きな方です。
英語力はどの程度必要ですか？ +
日本語メインのポジションの場合、日常会話レベル（TOEIC 500点程度）で応募可能です。 業務では主に日本語を使用しますが、社内コミュニケーションで英語を使う機会があります。 入社後も無料の英語学習プログラム（GoFluent）を利用できるので、働きながらスキルアップ可能です。
面接は何回ありますか？ +
通常、2回のオンライン面接を実施します。1次面接は採用担当者との面接（30-45分）、 最終面接はマネージャーとの面接（30-45分）です。すべてMicrosoft Teamsで実施し、 日本語メインで進行しますので安心してください。
応募から内定までどのくらいかかりますか？ +
順調に進めば、応募から内定まで約2-3週間です。書類選考に3営業日、 その後面接を2回実施し、最終面接から1週間以内に結果をお知らせします。

ビザ・移住について

就労ビザの取得は難しいですか？ +
TPが全面的にサポートしますので、ご安心ください。必要書類の案内から申請手続きまで、 専門チームが日本語で対応します。通常、内定から1-2ヶ月でビザが発行されます。 申請費用も会社が負担します。
渡航費用はどうなりますか？ +
日本からマレーシア/タイへの片道航空券を会社が支給します。 また、到着後6泊7日のホテル滞在も提供しますので、その間に住居を探すことができます。
家族を連れて行くことはできますか？ +
はい、可能です。就労ビザ（EP）取得後、配偶者や子供の帯同ビザ（DP: Dependent Pass）を 申請できます。日本人学校もあり、家族での移住も安心です。

仕事内容について

具体的にどんな仕事をしますか？ +
主に日本のお客様からの問い合わせに、電話・チャット・メールで対応します。 業界はEコマース、旅行、ホテル、金融、SNS、ITなど多岐にわたり、 商品の注文対応、予約変更、技術サポートなど、お客様の問題解決をサポートします。
勤務時間・シフトはどうなっていますか？ +
プロジェクトにより異なりますが、基本的には週5日勤務、1日8-9時間です。 日本時間に合わせたシフトが多く、早朝・夜間シフトもありますが、 シフト手当が付きます。土日祝日勤務の場合も手当があります。
在宅勤務は可能ですか？ +
職種とプロジェクトによりますが、ハイブリッド勤務（週2-3日在宅）が 可能なポジションもあります。詳細は面接時にご確認ください。

生活・住まいについて

住居探しはどうすればいいですか？ +
到着後6泊7日のホテル滞在中に探します。日本人スタッフが不動産エージェントを 紹介したり、物件探しのアドバイスをします。プール・ジム付きのコンドミニアムが 月5-8万円程度で借りられ、多くの方が1週間以内に決まります。
日本食は手に入りますか？ +
はい、充実しています。KLにはドン・キホーテ、AEON、伊勢丹があり、 バンコクにはフジスーパーがあります。日本食レストランも多数あり、 価格も日本より安価です。食生活で困ることはほとんどありません。
医療機関は安心できますか？ +
日本語対応可能な病院・クリニックが複数あり、医療水準も高いです。 会社の医療保険でカバーされるため、自己負担も最小限です。 歯科や専門医も充実しています。
治安は大丈夫ですか？ +
マレーシア・タイともに東南アジアの中では治安が良い国です。 特に日本人が多く住むエリアは安全で、女性の一人暮らしも問題ありません。 基本的な防犯意識を持っていれば、安心して生活できます。

福利厚生について

有給休暇はどのくらいありますか？ +
初年度は年間14日、勤続年数に応じて最大21日まで増えます。 これに加えて、病気休暇（年間14日）、マレーシア/タイの祝日、 一部の日本の祝日も休みとなります。
昇給・昇進の機会はありますか？ +
年2回の評価があり、パフォーマンスに応じて昇給します。 また、実力次第でチームリーダー、スーパーバイザー、マネージャーへの 昇進も可能です。年齢や勤続年数に関係なく、能力が評価されます。
退職時の帰国費用はどうなりますか？ +
契約期間満了または会社都合の退職の場合、帰国費用（航空券）が支給されます。 自己都合退職の場合は自己負担となります。詳細は雇用契約書でご確認ください。
